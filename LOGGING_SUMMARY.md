### 1. `app/logger.py` - Модуль логирования
Центральный модуль для настройки логирования с поддержкой двух языков.

- `auth_logger` - авторизация
- `chat_logger` - чат
- `friend_logger` - друзья
- `run_logger` - забеги
- `db_logger` - база данных

## Формат логов | Log Format

```
YYYY-MM-DD HH:MM:SS | LEVEL    | logger.name | Русский текст | English text: key=value, key2=value2
```

**Пример | Example:**
```
2026-05-09 03:20:15 | INFO     | gorun.auth | Успешный вход | Successful login: user_id=42, username=john_doe
```

## Преимущества | Benefits

1. **Двуязычность** - логи понятны русско- и англоязычным разработчикам
2. **Структурированность** - каждый лог содержит контекст (ID пользователей, объектов)
3. **Отладка** - легко отследить путь запроса через систему
4. **Мониторинг** - можно отслеживать активность пользователей
5. **Безопасность** - логируются попытки несанкционированного доступа
6. **Производительность** - можно найти узкие места по времени операций

## Использование | Usage

### Просмотр логов в реальном времени | Real-time Viewing
```bash
# PowerShell
Get-Content app\logs\gorun_20260509.log -Wait -Tail 50

# Bash
tail -f app/logs/gorun_20260509.log
```

### Поиск ошибок | Search for Errors
```bash
grep "ERROR" app/logs/gorun_20260509.log
grep "WARNING" app/logs/gorun_20260509.log
```

### Поиск по пользователю | Search by User
```bash
grep "user_id=42" app/logs/gorun_20260509.log
```

### Анализ активности | Activity Analysis
```bash
# Количество входов за день
grep "Successful login" app/logs/gorun_20260509.log | wc -l

# Количество созданных забегов
grep "Run successfully created" app/logs/gorun_20260509.log | wc -l
```
